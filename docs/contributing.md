# Contributing

* Star the project on [GitHub](https://github.com/dragosv/testcontainers-swift) and help spread the word :)
* Join our [Slack Workspace](https://slack.testcontainers.org/)
* Post an issue if you find any bugs
* Contribute improvements or fixes using a Pull Request. If you're going to contribute, thank you! Please just be sure to:
    * Discuss with the authors on an issue ticket prior to doing anything big.
    * Follow the style, naming, and structure conventions of the rest of the project.
    * Make commits atomic and easy to merge.
    * Verify all tests are passing with `swift test` (requires Docker running).
    * Run `swiftformat --lint .` to check formatting before submitting.
